# Personal Portfolio Website in AWS

A responsive personal portfolio website built using HTML, CSS, and JavaScript and deployed on Amazon S3 with Amazon CloudFront for secure, fast, and globally distributed content delivery.

## 🌐 Live Demo

https://d3uol8l4nn6ofa.cloudfront.net/

## ✨ Features

- Responsive Design
- About Me
- Skills
- Projects
- Contact Page
- Resume
- AWS Cloud Deployment

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Amazon S3
- Amazon CloudFront
- Git
- GitHub

## ☁ AWS Architecture

User
↓
CloudFront
↓
Amazon S3
↓
HTML, CSS, JavaScript, Images

## 🚀 Deployment

- Hosted on Amazon S3
- Delivered through Amazon CloudFront
- Configured `index.html` as the Default Root Object
- Used CloudFront cache invalidation after deployment
